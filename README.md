# Reedly

[![Build Status](https://travis-ci.org/lucerion/reedly.svg?branch=master)](https://travis-ci.org/lucerion/reedly)

Stand-alone RSS aggregator


## Setup

    mix deps.get


## Dependencies

  * erlang-dev
  * erlang-dialyzer
  * erlang-tools
  * erlang-xmerl


## Development

### Type checker

    mix dialyzer

### Static code analyzer

    mix credo

### Code formatter

    mix format
